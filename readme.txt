== Description ==

An example theme illustrating content protection in templates, a Twenty Twenty Child Theme.

This child theme requires [Groups](https://wordpress.org/plugins/groups/) and the [Twenty Twenty](https://wordpress.org/themes/twenty-twenty/) parent theme.

For details on using Groups please refer to its [Documentation](http://docs.itthinx.com/document/groups/) pages.
